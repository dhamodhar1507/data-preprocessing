# Data Preprocessing Assignment

This project demonstrates a complete data preprocessing pipeline using Python.

## Steps Performed

1. Handling missing values using median and mode
2. Removing duplicate records
3. Encoding categorical variables using Label Encoding and One-Hot Encoding
4. Feature scaling using Min-Max Scaling and Standardization

## Files Included

- data_preprocessing.ipynb : Jupyter notebook with complete implementation
- README.md : Description of the project
